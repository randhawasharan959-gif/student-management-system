# Student Management System

A simple web app to manage student records.

## Features
- Add new students
- View all students in a table
- Search students by name, roll, class, or section
- Edit student details
- Delete students

## How to Use
1. Open `index.html` in your browser.
2. Navigate to "Add Student" to enter new student info.
3. Go to "View Students" to see, search, edit, or delete records.

## Data Storage
Student data is stored locally in your browser's localStorage. It persists across sessions but is not shared between devices or users.

## Deployment on GitHub Pages
This app is designed to work on GitHub Pages as a static site. Simply push the HTML, CSS, and JS files to your GitHub repository and enable Pages in the repository settings.
